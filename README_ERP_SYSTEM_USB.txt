🎯 نظرة عامة | Overview
────────────────────────────────────────────────────────────────────────────

هذا هو نظام DED ERP الكامل المحمول الذي يمكن تشغيله من أي جهاز كمبيوتر
يحتوي على Python دون الحاجة لتثبيت أي شيء على الجهاز المضيف.

This is the complete portable DED ERP System that can run from any computer
with Python installed without needing to install anything on the host machine.

================================================================================

📋 المتطلبات | Requirements
────────────────────────────────────────────────────────────────────────────

✅ Python 3.7 أو أحدث | Python 3.7 or newer
✅ متصفح ويب (Chrome, Firefox, Edge, إلخ) | Web browser
✅ 50 MB مساحة حرة على الأقل | At least 50 MB free space

🚀 كيفية الاستخدام | How to Use
────────────────────────────────────────────────────────────────────────────

الطريقة 1: التشغيل السريع (موصى بها) ⭐
Method 1: Quick Start (Recommended) ⭐

1. أدخل الفلاشة في أي جهاز كمبيوتر
   Insert the USB drive into any computer

2. افتح الفلاشة (D:\ أو E:\ أو F:\ حسب الجهاز)
   Open the USB drive (D:\ or E:\ or F:\ depending on the computer)

3. انقر نقراً مزدوجاً على:
   Double-click on:
   
   ⭐ START_DED_ERP_SYSTEM_PORTABLE.bat

4. سيتم تلقائياً:
   The system will automatically:
   
   ✅ التحقق من وجود Python
      Check for Python installation
   
   ✅ تثبيت المكتبات المطلوبة
      Install required packages
   
   ✅ تشغيل نظام ERP
      Start the ERP system
   
   ✅ فتح المتصفح على http://127.0.0.1:5000
      Open browser at http://127.0.0.1:5000

────────────────────────────────────────────────────────────────────────────

الطريقة 2: التشغيل اليدوي
Method 2: Manual Start

1. افتح موجه الأوامر (Command Prompt)
2. انتقل إلى مجلد التطبيق:
   Navigate to the app folder:
   
   cd D:\DED_ERP_System_Complete
   
3. شغل التطبيق:
   Run the application:
   
   python run.py

🔧 استكشاف الأخطاء | Troubleshooting
────────────────────────────────────────────────────────────────────────────

❌ المشكلة: "Python is not installed"
✅ الحل: قم بتثبيت Python من https://www.python.org/downloads/

❌ المشكلة: "Port 5000 is already in use"
✅ الحل: أغلق أي تطبيق آخر يستخدم المنفذ 5000

❌ المشكلة: "Database is locked"
✅ الحل: تأكد من عدم تشغيل نسخة أخرى من التطبيق

❌ المشكلة: لا يفتح المتصفح تلقائياً
✅ الحل: افتح المتصفح يدوياً واذهب إلى http://127.0.0.1:5000
